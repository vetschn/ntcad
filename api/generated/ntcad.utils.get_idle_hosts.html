ntcad.utils.get_idle_hosts#

utils.get_idle_hosts(prefix)#

Finds all hosts that are currently idle.

Invokes the cload command to find all hosts that are currently idle.

A host is considered idle if all three load averages are below 0.5.

Parameters:
prefixstr

The prefix of the hosts to search for.

Returns:
idle_hostslist[str]

A list of all hosts that are currently idle.

Examples

Check which nodes are currently idle:

>>> get_idle_hosts("node") 
["node1", "node2", "node3"] # NOTE: This may take a while.
